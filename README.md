# Snake
It's terminal snake game

## How install
1. Move to home directory: cd
2. Download: git clone git@github.com:Chimi70/Snake.git
3. move to snake folder: cd snake
4. Compile: gcc snake.cpp -o snake
5. Run Game: ./snake

## How uninstall
1. Move to home directory: cd
2. Remove game: rm snake snake.cpp
